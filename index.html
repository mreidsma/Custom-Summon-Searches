Create Custom Search

Serials Solutions has a powerful tool for building scoped searches. Use it if you can embed Javascript on your webpage. If not, this tool creates the same search with good old HTML.

Summon searches nearly all of our holdings by default. To create a more targeted search, select limiters below and copy the HTML from the bottom of the page to insert into any website.

Copy this HTML:

<!-- Start of Custom Summon Search code -->
<form method="GET" action="http://gvsu.summon.serialssolutions.com/search">
<input type="text" name="s.q" />
<input type="hidden" name="spellcheck" value="true" />
<input type="hidden" name="keep_r" value="true"/>
<input type="submit" value="Search" />
</form>
<!-- End of Custom Summon Search code -->
